## License of Contributions

All contributions to this repository are accepted under the same license as the project itself: GNU GPL v3.0.

Please do not submit code, data, text, or other material that is incompatible with the project license.

## Community Extensions and Step Packages

CogniFlow supports extension through step packages and related modules. Because such packages may be tightly integrated with the Cogniflow core, ontology, and pipeline engine, contributors should ensure that their licensing model is compatible with GPLv3 where required.

If you are unsure whether a planned package or dependency is license-compatible, please open an issue before submitting a pull request.
